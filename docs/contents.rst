****************************
Opexebo: Neurology in Python
****************************

Opexebo is a collection of code used in the Moser group in the Kavli Institute for Neuroscience (https://www.ntnu.edu/kavli). It is intended to provide a modular, extensible collection of specialised methods for processing and analysing behavioural electrophysiology data.
